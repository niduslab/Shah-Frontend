## 🚀 Quick Start

### Create a Brand Page
```
1. Go to /admin/page-templates
2. Click "Brand Page"
3. Fill page title and slug
4. Add sections from grid
5. Click section to edit
6. Fill content fields
7. Adjust settings
8. Click "Save Page"
```

### Edit Existing Brand Page
```
1. Go to /admin/pages
2. Click "Edit" on brand page
3. Click section to edit
4. Modify content
5. Click "Save Changes"
```

## 🎨 Design Tips

### Images
- Hero: 1920x800px
- Categories: 600x600px
- Products: 800x800px
- Collage: 400x400px

### Text Length
- Headings: 5-10 words
- Descriptions: 15-30 words
- Buttons: 1-3 words

### Colors
- Dark backgrounds for categories
- Light text on dark images
- Orange (#FF6F00) for CTAs

## 🔧 Troubleshooting

### Section Not Showing Content
- Check if section is active
- Verify all required fields filled
- Check image URLs are valid

### Images Not Displaying
- Verify image URL is correct
- Check image file exists
- Try re-uploading image

### Save Button Not Working
- Fill page title and slug
- Add at least one section
- Check browser console for errors

## 📱 Responsive Behavior

| Device | Layout |
|--------|--------|

| Tablet (768-1199px) | 2-column, some stacking |
