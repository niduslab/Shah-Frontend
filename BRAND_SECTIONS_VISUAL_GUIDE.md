## Responsive Behavior

### Desktop (1200px+)
- Full width layouts
- Multi-column grids
- Side-by-side content

### Tablet (768px - 1199px)
- 2-column grids
- Stacked product cards
- Reduced spacing

### Mobile (< 768px)
- Single column
- Stacked sections
- Full-width images
- Larger touch targets

## Color Schemes

### Dark Theme (Default for Categories)
```
Background: #1a1a1a (dark gray/black)
Text: #ffffff (white)
Accent: #FF6F00 (orange)
```

### Light Theme
```
Background: #ffffff (white)
Text: #1a1a1a (dark gray)
Accent: #FF6F00 (orange)
```

### Gray Theme
```
Background: #f3f4f6 (light gray)
Text: #1f2937 (dark gray)
Accent: #FF6F00 (orange)
```

## Typography

### Headings
- Main Heading: 48px, Bold
- Section Title: 36px, Bold
- Card Title: 24px, Semibold

### Body Text
- Description: 16px, Regular
- Button Text: 14px, Medium
- Label Text: 14px, Regular

## Spacing

### Section Padding
- Top/Bottom: 80px
- Left/Right: 40px (desktop), 20px (mobile)

### Card Spacing
- Gap between cards: 24px
- Internal padding: 24px

### Element Spacing
- Heading to description: 16px
- Description to button: 24px
- Between form fields: 12px

## Best Practices

### Image Sizes
- Hero backgrounds: 1920x800px
- Category images: 600x600px
- Product images: 800x800px
- Collage images: 400x400px

### Content Length
- Headings: 5-10 words
- Descriptions: 15-30 words
- Button text: 1-3 words

### Link Structure
- Categories: `/category/slug`
- Products: `/product/slug`
- Pages: `/page-slug`

### Accessibility
- Alt text for all images
- Sufficient color contrast
- Keyboard navigation support
- Screen reader friendly
