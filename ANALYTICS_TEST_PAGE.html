Backend Status Check

Test if the backend analytics API is accessible:

1. Test Page View Tracking

Tracks when a page is viewed

2. Test Product View Tracking

Tracks when a product is viewed

3. Test Cart Event Tracking

Tracks cart actions (add, update, remove)

4. Test Checkout Tracking

Tracks checkout funnel stages

5. Test Search Tracking

Tracks search queries

📋 Instructions

  1. Make sure your Laravel backend is running on http://localhost:8000
  2. Make sure you've run php artisan migrate to create analytics tables
  3. Click the buttons above to test each tracking endpoint
  4. Check the results below each button
  5. Open browser DevTools (F12) → Network tab to see requests
